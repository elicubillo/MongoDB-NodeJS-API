# MongoDB-NodeJS-API

Example of how to interact with the MongoDB database using an API developed in NodeJS

## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Install mongoose

```bash
npm install mongoose --save
```

3. Run the application
